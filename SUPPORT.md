# 支持说明

本项目公开的是 ESP32-S3 三色墨水屏终端固件、Web 管理端、文档和免编译烧录包。支持重点是帮助复刻用户完成烧录、配网、屏幕选择、基础排障，以及帮助二次开发者理解工程结构。

## 支持渠道

- Gitee Issue：适合提交可复现 bug、文档问题、屏幕适配反馈和功能建议。
- QQ 交流群：适合交流复刻经验、硬件选择、烧录和实机显示问题。
- README 中的维护者联系方式：适合商业授权、安全问题或不适合公开贴出的脱敏沟通。

## 提交问题前

请先确认：

- 已阅读 `README.md` 的硬件要求、快速开始和分区表说明。
- 已阅读 `release/README.md` 的整包固件烧录方式。
- 已在设备配置页选择正确屏幕型号，并保存后重启。
- 如使用低功耗，已区分正常开机、按键唤醒和 RTC 定时唤醒。

## 推荐提供的信息

为了减少来回确认，请尽量提供：

- 固件版本：网页 `/version` 或串口 `App version`。
- 固件提交号或使用的 release 文件。
- ESP-IDF 版本，若是自行编译。
- 开发板型号、Flash/PSRAM、屏幕尺寸、驱动 IC、BUSY 空闲电平。
- 配置页选择的面板名称。
- 从上电到问题出现的关键串口日志。
- 屏幕照片、网页截图或相关配置截图。

涉及 WiFi 名称、天气 API Key、Codex/中转站 API Key、局域网 IP、地理位置和个人信息时，请先脱敏。

## 常见支持边界

| 类型 | 支持方式 |
|------|----------|
| 免编译烧录失败 | 优先按 `release/README.md` 检查下载文件、SHA256、串口、BOOT 键、Flash Size 和是否需要 ERASE |
| 自行编译失败 | 优先确认 ESP-IDF v5.5.x、目标芯片 `esp32s3`、是否清理旧 `build/` 和本地 `sdkconfig` |
| 屏幕花屏/偏移/颜色异常 | 提供屏幕型号、驱动 IC、BUSY 电平、配置页面板名和照片；先验证清屏、纯黑、纯红、边框和棋盘格 |
| 字体或 UI 显示问题 | 提供屏幕尺寸、页面模式、照片和输入文本；说明是否使用 Web 画板字体渲染 |
| 低功耗问题 | 提供低功耗开关状态、唤醒来源、当前显示模式和串口日志 |
| 新屏幕适配 | 优先提供供应商资料、FPC 标识、驱动 IC、初始化资料和实机测试图 |

## 不承诺的内容

- 不保证所有第三方屏幕批次都能直接使用同一驱动配置。
- 不保证未主测面板的局刷、快刷、灰阶或低温刷新效果。
- 不保证外部 API 服务可用性，例如天气接口或中转站额度接口。
- 不提供公网部署安全加固承诺；设备建议运行在可信局域网内。
- 非商业许可项目不默认提供商业量产支持，商业使用需另行取得授权。
