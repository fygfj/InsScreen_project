# 喵哎-MiaooAim 使用说明书

## 当前版本提示（2026-06-28）

当前版本已覆盖图片、天气、时钟、日历、课程表、待办、倒计时、Codex 额度看板、留言板、画布、OTA 和低功耗，并增强了 Web 画板字体渲染、外置 fontfs 字库、日历跨日刷新、5.83 寸页面适配和低功耗按模式唤醒。首次使用后建议在设备配置页设置管理密码；启用 Basic Auth 后，图片、配置、课程、待办、倒计时、额度看板、画布等数据/媒体接口也会要求登录。

已增强：上传图片不会再被旧图库请求抢屏；日历、画布、留言等网页推屏会等本次渲染结束再反馈，若被新操作替换会提示 `canceled=true`；面板型号保存后需重启生效；SPIFFS 挂载失败不会自动清空用户内容，正常启动会保留 AP/Web 诊断入口，配置页可重试挂载或输入确认口令后格式化；若屏幕不可用，也可直接用默认热点密码进入网页恢复配置；配置备份/恢复不会包含管理账号密码；串口日志不输出 AP 密码。

> 适用于基于 ESP32-S3 的三色墨水屏终端（产品名喵哎-MiaooAim，固件代号 `epaper_uploader`，与仓库 `README.md` 描述一致）。<br>

## 1. 产品能做什么

- **看时间、万年历**：时钟表盘、月历（含农历、节气、节日等，以固件与配置为准）。
- **看天气**：接入和风天气后，可显示实况与多日预报（需您在网页里配置 API）。
- **看课程表、待办、倒计时**：在浏览器里编辑，屏幕按当前模式展示。
- **看 Codex/中转站额度**：在设备配置页填写额度 API URL 与 API Key 后，可显示余额、已用、今日用量、请求数和令牌数。
- **看照片与轮播**：上传 JPEG / PNG / BMP，可单张展示或按间隔自动轮播。
- **留言与画布**：文字留言板、可视化画布排版（网页编辑，推送到屏）。
- **省电**：可开启闲置后深度睡眠，到点或用按键唤醒（见下文）。

墨水屏刷新较慢（约十余秒一次全屏刷新属正常现象），且刷新期间不宜连续猛按按键。

## 2. 使用前准备

1. **供电**：按您购买的成品说明连接锂电池或 USB 供电；电量过低时可能影响 WiFi 与刷新稳定性。  
2. **WiFi 环境**：首次配网需要一部手机或电脑；日常建议设备与您的手机、电脑在同一局域网，便于用浏览器管理。  
3. **浏览器**：推荐使用 Chrome、Edge、Safari 等现代浏览器访问设备网页。

---

## 3. 第一次开机与配网

1. **上电**后，设备会开启 WiFi 热点，名称一般为 **`ESP32_EPD_xxxxxx`**（`xxxxxx` 为设备后缀），默认密码为 **`12345678`**。
2. 用手机或电脑**连接该热点**。  
3. 打开浏览器，访问 **`http://192.168.4.1/`**（部分环境也可试 `http://192.168.4.1` 不带末尾斜杠）。  
4. 进入 **「设备配置」**（或项目中的 `config.html` 页面），按提示选择并输入您家中的 **WiFi 名称与密码**，保存后设备会尝试连接路由器。  
5. 连接成功后，请改用下面 **「日常访问方式」** 打开网页，不要长期停留在热点页面。

> 若您的路由器支持 mDNS，设备会以类似 **`http://epdxxxxxx.local/`** 的主机名出现在局域网中（具体以串口日志或路由器客户端列表为准）。

---

## 4. 日常如何打开管理页面

在设备已连上您家 WiFi 的前提下，任选其一：

| 方式 | 说明 |
|------|------|
| **mDNS** | 浏览器访问 `http://epdxxxxxx.local/`（与启动日志中提示一致） |
| **IP 地址** | 在路由器管理界面或串口日志中查看设备 IP，访问 `http://设备IP/` |
| **未连上路由器时** | 再次连接热点 `ESP32_EPD_xxxx`，访问 `http://192.168.4.1/` |

首页一般为 **`/`** 或 **`index.html`**。当前 Web 已拆分为多个页面：画廊 `/gallery`、天气 `/weather`、时钟 `/clock`、日历 `/calendar`、留言 `/message`、额度 `/codex`、课程表 `/timetable`、待办 `/todo`、倒计时 `/countdown`、画布 `/board`、系统配置 `/config`。

部分敏感操作（如上传、删除、OTA 等）在网页中配置 **Basic Auth（用户名/密码）** 后，浏览器会提示输入账号密码，请在 **设备配置** 中预先设置并牢记。配置导出/导入不会备份管理密码，恢复备份后如需访问保护请重新设置。

## 5. 机身按键说明

本机为三键设计（丝印多为 **SW3 / SW4 / SW5**，按下为高电平）。在固件中的默认含义为：

| 按键 | 作用 |
|------|------|
| **SW3** | **上一个**显示模式（在模式列表中向前切换） |
| **SW4** | **刷新当前模式**（再执行一次当前模式的显示逻辑；在**轮播**模式下效果为切换到下一张图，见下节说明） |
| **SW5** | **下一个**显示模式 |

**模式顺序**（与固件注册顺序一致，循环切换）：

1. 时钟 → 2. 日历 → 3. 课程表 → 4. 天气 → 5. 轮播图 → 6. 待办 → 7. 倒计时 → 8. 额度 → 再回到时钟 …

**留言板、画布、单张图片上传**等由网页配置并推屏，**不在这 8 档按键循环里**；请从首页或导航进入「留言 / 画布 / 上传」等页面操作。

**关于 SW4「刷新」**：  
- 在时钟、日历、课程表等模式下，一般为用当前数据**重新画一屏**。  
- **天气**模式下，一般为用**已有缓存**重画；**不会**仅因按一下键就自动去网络拉取最新预报（更新间隔由网页里天气配置决定）。  
- **轮播**模式下，当前固件将「刷新」与「下一张图」绑在同一套逻辑上，按 SW4 往往会**切到下一张**画廊图片。

刷新过程中日志可能出现 **display busy，按键忽略**，属保护机制，请等本轮刷新结束后再按。

## 6. 图片、画廊与轮播

- 在网页 **上传** 入口选择 **JPEG / PNG / BMP**，设备会转换并显示在墨水屏上（三色抖色、拉伸全屏等由固件自动处理）。  
- **画廊 / 轮播**：在配置中开启轮播、设置间隔与顺序/随机等选项后，设备会按间隔自动切换图片；也可用按键切换到「轮播图」模式后配合 SW4 手动切图。  
- 具体菜单名称以您设备上的网页为准（如「画廊」相关页面）。

---

## 7. 天气

- 使用 **和风天气（QWeather）** 类接口时，需在 **设备配置 / 天气配置** 中填写 **API Key、城市或 Location ID、API 主机** 等（以网页表单为准）。  
- 保存后需等待同步成功；若从未成功拉取过数据，天气模式可能提示无缓存，属正常现象，请检查网络与 API 配置。

---

## 8. 低功耗 / 深度睡眠（若网页提供）

在 **设备配置** 中可开启低功耗相关选项（具体文案以网页为准），典型行为包括：

- 一段时间**无操作**（无 HTTP 访问、无按键等，以配置为准）后进入 **深度睡眠**，以延长电池续航。  
- 可由 **定时器** 到期唤醒；**按键唤醒**在 ESP32-S3 上仅 **SW3、SW5（GPIO 9 与 3）** 有效（芯片 EXT1 只支持部分 RTC 引脚）。**中间键 SW4（GPIO46）不能从深睡唤醒**，上电运行期间仍可用。  
- 正常上电、复位、按键唤醒时仍会启动 AP+STA、mDNS 和网页后台，方便继续管理设备。开启低功耗按钮不会关闭后台，只会把运行期 WiFi 调到省电模式。
- 只有 **定时器自动唤醒** 才会走 **快速刷新** 路径：只更新当前记忆的模式内容，然后回到睡眠。图片、轮播、待办、倒计时、课程表等本地页面通常不启 WiFi；天气、额度、带天气摘要或时间无效需要校时的页面才短时间启用 STA WiFi。

### 存储空间怎么看

配置页或上传页里的 SPIFFS 进度条表示 **用户文件区已用比例**，不是剩余比例。当前默认分区中用户 SPIFFS 为 3.875MB，用于图片、画布素材和运行时文件；`fontfs` 是独立的 2.875MB 字库分区，不占用用户图片空间。

若您希望设备长期当「常亮小主机」用，可关闭或放宽睡眠相关设置。

## 9. 固件升级（OTA）

在网页中若提供 **OTA / 固件升级** 入口，上传官方提供的固件包即可（通常需先配置 Basic Auth）。升级过程中请勿断电；失败时可按开发文档用 USB 重新烧录。

## 10. 常见问题

| 现象 | 建议 |
|------|------|
| 找不到网页 | 确认手机/电脑与设备同网；试 IP；未连路由器时连热点 `192.168.4.1` |
| 刷新很慢 | 墨水屏物理特性，全屏刷新常需十余秒，请耐心等待 |
| 按键没反应 | 可能正在刷新，稍后再试；或检查供电与固件是否正常 |
| 天气不显示 | 检查 WiFi、API Key、城市 ID、HTTPS 是否可用 |
| 颜色与电脑不一致 | 三色屏通过抖色模拟灰阶，属正常现象 |
| 网页显示 `canceled=true` | 本次推屏被更新的上传、按键或其它网页操作替换，通常只需以最后一次操作为准 |
| 屏幕没初始化但网页能打开 | 这是诊断保护：请检查屏幕排线、BUSY 电平、面板型号和供电；网页可用于重新配置或导出信息 |
| 屏幕不显示但需要进网页 | 直接连接热点 `ESP32_EPD_xxxxxx`，默认密码 `12345678`，打开 `192.168.4.1/config` 切换面板并重启 |
| 提示 SPIFFS 异常 | 若屏幕显示 `文件系统恢复`，按页面上的热点和密码连接后打开 `192.168.4.1/config`；先点“重试文件系统”，若确认不需要保留图库/画布素材，再输入 `FORMAT_SPIFFS` 执行格式化 |

## 11. 安全与隐私提示

- 热点默认密码为 **`12345678`**，用于降低首次配网和屏幕异常时的售后门槛；配网后仍建议在 **设备配置** 中设置管理用户名与密码。
- **管理密码（Basic Auth）**：在**未设置**时，固件为便于首次配网，对依赖鉴权的**写操作**通常不强制登录；同一路由器/热点下的其他设备在理论上也可调用这些接口。配网完成后请尽快在 **设备配置** 中设置用户名与密码。设置成功后，敏感操作会要求浏览器凭据。  
- **OTA**：未设置管理密码时，通过网页刷写固件会被拒绝，需先配置 Basic Auth 或使用 USB 烧录。  
- 天气 API Key 等属于个人凭证，请勿截图公开分享；设置成功后，数据/媒体类接口也会要求登录；页面本身可能仍可打开以触发浏览器认证流程。请避免在不可信网络长期暴露设备。

## 12. 规格与版本（参考）

- **主控**：ESP32-S3；**显示**：常见为 4.2"（400×300）或 5.83"（648×480）三色墨水屏，以实际硬件与配置为准。  
- **5.83 寸屏幕**：若供应商标注兼容“微雪 5.83b V2 / 648×480 / BWR”，配置页优先选择 **微雪 5.83 寸黑白红 B V2**；若画面铺满且黑红颜色正常，边缘模糊多半优先检查屏体、FPC 接触、供电和温度。
- **软件版本**：以设备网页 **`/version`** 或串口启动日志中的版本号为准；当前工程版本为 v2.3.4。
