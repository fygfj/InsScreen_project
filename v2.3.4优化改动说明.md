# v2.3.4 优化改动说明

本文面向复刻用户和二次开发者，概述 v2.3.4 中已经合入的主要优化。更细的工程实现请参考 `ENGINEERING_DOCS.md`、`IMPLEMENTATION.md` 和 `docs/fonts.md`。

## 版本定位

v2.3.4 在 v2.3.3 的公开发布整理基础上，继续补充 4.2 寸屏幕参考驱动选项，并修复低功耗模式下后台轮询和自动刷新影响深睡的问题。

## 屏幕驱动选项

- 配置页新增 UC 4.2 寸黑白红三色参考驱动选项，对应 `EPD_PANEL_REF_UC8176_42_BWR`。
- 配置页新增 SSD1619A 4.2 寸黑白双色参考驱动选项，对应 `EPD_PANEL_REF_SSD1619_42_BW`。
- 同步后台返回的屏幕名称、枚举注释和驱动表名称。

说明：本版本补充的是参考驱动入口和配置选项，并未重写底层初始化或刷新时序。新增面板仍建议先验证清屏、纯黑、纯红、边框、棋盘格和文字基线。

## 低功耗修复

- 自动刷屏完成后不再重置闲置计时，只更新最近 EPD 刷新时间。
- HTTP 后台状态轮询不再被当作用户活跃操作，避免 `/status`、`/wifi_status` 等周期请求持续阻止深睡。
- 新增用户操作记录入口，打开页面、保存配置、上传图片、点击显示和删除等真实操作仍会刷新闲置计时。
- 用户停留在后台页面但不操作时，状态轮询不应阻止设备按配置进入深度睡眠。

- 同时保留带版本后缀的 `release/epaper_uploader_full_16MB_v2.3.4.bin` 与 ZIP，便于归档和人工核对。
