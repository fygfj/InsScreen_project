# v2.3.5 优化改动说明

本版本是在温湿度、蜂鸣器、SD 卡和启动显示稳定性基础上的整理版本。版本号已更新为 `2.3.5`，网页端通过 `/version` 与 `/status` 读取固件版本，因此配置页和上传页会同步显示新版本。

## 启动显示

- 启动欢迎页标题改为 `Ink Screen_Cy`。
- 优化上电显示流程：先显示欢迎页，等待约 10 秒，再恢复上次保存的界面。
- 新增启动期显示保护，欢迎页与恢复上次界面期间，天气、时钟、日历、轮播等后台自动刷新不会抢屏。
- 解决启动阶段可能出现的显示请求互相覆盖，降低上电乱刷、半屏、白屏风险。
- 启动日志已验证流程：欢迎页显示完成后，按保存模式恢复到天气或轮播等上次界面。

## 固件版本与网页同步

- 项目版本号从 `2.3.4` 更新为 `2.3.5`。
- 启动页底部版本信息使用固件版本号。
- 网页端固件版本由后端接口读取，无需单独改网页硬编码。
- `/version`、`/status` 返回的版本均会显示 `2.3.5`。

## AHT20 温湿度

- 加入 AHT20 温湿度传感器驱动和本地传感器管理逻辑。
- I2C 使用 `SDA GPIO1`、`SCL GPIO2`，地址 `0x38`。
- 支持检测模块是否存在、读取温度、湿度、数据年龄和最后错误状态。
- 网页配置页增加本地温湿度状态与开关配置。
- 支持配置是否显示到时钟页、天气页、日历页。
- 已调整温湿度显示布局：
  - 天气界面放到三日预报那一行。
  - 时钟界面放到底部一行。
  - 日历界面与时钟界面一致，放到底部。
- 上传图片页面在内存显示旁边增加温湿度状态格。
- 低功耗路径中考虑传感器和 I2C 总线状态，避免休眠时额外增加不必要功耗。

## 蜂鸣器

- 加入蜂鸣器驱动，输出引脚为 `GPIO17`。
- 网页配置页增加蜂鸣器设置。
- 支持声音总开关和音量参数设置。
- 支持以下提示音开关：
  - 开机就绪。
  - 实体按键确认。
  - Wi-Fi 配网成功。
  - Wi-Fi 配网失败。
  - 低电量按键提醒。
  - OTA 升级结果。
  - 显示错误提示。
- 成功提示音采用短促两声。
- 失败提示音使用可区分的错误节奏。
- 低电量提醒按需求改为按键操作时触发，低电量时快响 5 下，不做持续报警。

## SD 卡基础能力

- 加入 SD 卡电源控制，默认断电以降低待机和休眠功耗。
- SD 卡使用 SDMMC 1-bit 模式，当前占用：
  - `CLK GPIO48`
  - `CMD GPIO47`
  - `D0 GPIO41`
  - `PWR_EN GPIO21`，低电平使能
- 未插卡时不会影响其他功能，只是 SD 相关功能不可用。
- 支持网页检测 SD 卡、挂载、卸载和断电。
- 修复反复检测/卸载后可能出现 `ESP_ERR_NO_MEM` 的问题，VFS 不再重复注册。
- 支持读取 SD 卡总容量、剩余容量、卡名和目录状态。
- 修复剩余容量显示问题，使用 FatFS `f_getfree()` 获取容量数据。

## SD 卡目录与文件功能

- SD 卡挂载后会自动准备项目目录：
  - `/sdcard/epd`
  - `/sdcard/epd/images`
  - `/sdcard/epd/backup`
  - `/sdcard/epd/logs`
- 上传图片页面增加 SD 图片区。
- 支持列出 SD 卡图片。
- 优化上传页面“刷新SD图片”的操作反馈，点击后主日志和 SD 状态区都会立即提示。
- SD 图片列表读取增加超时提示，避免 SD 卡异常时网页看起来没有反应。
- 增加 SD 图片网页缩略图预览，使用流式读取，不在 ESP 端解码整图。
- SD 缩略图默认只加载前 8 张，降低网页打开时的 SD 读压力和浏览器内存占用。
- 支持从 SD 卡导入图片到内部 SPIFFS 图库。
- 支持把内部图库图片备份到 SD 卡。
- 支持单个导入、批量导入、单个备份、批量备份。
- 配置页支持将当前配置备份到 SD 卡。
- 配置页支持从 SD 卡恢复配置。
- 配置备份路径为 `/sdcard/epd/backup/epd_config_backup.json`。
- 备份和导出动作不主动重启设备。

## 诊断日志导出

- 加入诊断日志缓存模块。
- 固件运行时保留最近日志环形缓存，缓存大小为 `16 KB`。
- 原串口日志输出不受影响。
- 配置页增加“导出诊断日志”按钮。
- 点击后才会挂载并写入 SD 卡，不会平时持续写卡。
- 诊断日志保存到 `/sdcard/epd/logs/`。
- 日志文件包含：
  - 当前时间与同步状态。
  - 固件版本、IDF 版本、运行分区。
  - 堆内存状态。
  - 电池电压、电量、充电状态。
  - AHT20 状态和读数。
  - SD 卡挂载、容量和错误状态。
  - 最近串口日志。
- 未插卡时导出会提示失败，但不影响其他网页和设备功能。

## 网页端优化

- 配置页增加本地温湿度、蜂鸣器、SD 卡、诊断日志相关入口。
- 上传图片页面增加温湿度状态显示。
- 上传图片页面增加 SD 图片导入/备份操作区。
- 状态接口增加 SD 卡容量、剩余容量、目录状态等信息。
- 网页端固件版本显示跟随固件版本号 `2.3.5`。

## 低功耗相关

- SD 卡默认不上电，只有检测、导入、备份、日志导出等需要 SD 的操作才上电挂载。
- SD 卡卸载后会关闭电源并释放相关 IO 状态。
- 未插 SD 卡不会阻塞普通显示、网页、传感器和按键功能。
- AHT20 与 I2C 相关逻辑已纳入休眠/低功耗路径考虑。

## 已知注意项

- 当前启动日志仍能看到 EPD `BUSY timeout`，驱动会进入固定等待兜底模式。
- `BUSY timeout` 与本次 SD、AHT20、蜂鸣器和启动逻辑优化无直接关系。
- 如果后续仍偶发白屏、花屏或刷新时间异常，优先检查墨水屏 BUSY 引脚电平配置、硬件连接和屏幕驱动参数。
