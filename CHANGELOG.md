# 更新日志

本文记录公开仓库中对用户和复刻者有影响的主要变化。

## 2026-06-28

### 修复与发布准备

- 将源码版本提升到 v2.3.4，并重新构建对应的 16MB 整包固件。
- 配置页补充 UC 4.2 寸三色参考驱动和 SSD1619A 4.2 寸双色参考驱动选项，便于复刻用户选择相近面板路径。
- 修复低功耗模式下网页状态轮询和自动刷屏持续刷新闲置计时的问题，避免设备因后台轮询或自动页面刷新而无法按预期进入深度睡眠。

## 2026-06-26

### 修复与发布

- 修复 4.2 寸经典日历页顶部年支信息与标题分隔线过近、底部网格线与页脚分隔线重叠的问题，重新规划标题、星期、网格和页脚区域坐标。
- 按最新源码重新生成 `release/epaper_uploader_full_16MB.bin` 和 `release/epaper_uploader_full_16MB.zip`，并同步更新 SHA256 校验信息。

## 2026-06-25

### 修复

- 优化 4.2 寸欢迎页和日历页小字号显示：新增共享 1bit 清晰位图表，欢迎页标题/网络信息/使用步骤、普通日历和带天气日历的小字不再直接使用圆润 TTF 内置小字，减少发虚和模糊感；5.83 寸布局和时钟/天气数码管风格不受影响。

### 文档同步

- 重排 README 首页首屏，新增“一分钟了解项目”和“快速体验”入口，把免编译烧录、首次配置、二次开发、硬件复刻和屏幕适配路径前置，减少 Gitee 自动目录里的零散状态项。
- 扩充 `docs/gitee-about.md`，提供 Gitee 右侧简介、项目定位、发布状态、超短标题和首页首屏建议，便于维护公开仓库展示内容。
- 新增 `CODE_OF_CONDUCT.md`，并接入 README、文档索引、贡献指南和 Gitee Issue / PR 模板，补齐公开仓库协作规范、日志脱敏和维护者处理边界。
- 同步低功耗 WiFi 策略：正常上电、复位、按键唤醒保持 AP+STA、mDNS 和 HTTP 后台；只有 RTC 定时器唤醒进入 quick-refresh，并按当前显示模式决定是否启动 STA-only WiFi。
- 同步 5.83 寸屏幕说明：标注兼容微雪 5.83b V2 的 648×480 BWR 屏，配置页优先选择“微雪 5.83 寸黑白红 B V2”；若画面铺满且颜色正常，边缘模糊优先排查屏体、FPC、供电和温度。
- 同步 Web 多页面入口、73 条 HTTP 路由、12 个 HTML + 1 个 PNG 嵌入资源、`fontfs=0x940000`、用户 SPIFFS 3.875MB 和 release 重新构建状态。
- 修正专题文档中过期的存储容量、HTTP 路由数量和分区说明。
- 扩展 README 首页“项目状态”区域和 `docs/gitee-about.md` 仓库简介建议，让 Gitee 首页第一屏更清楚说明当前版本、交付内容、适用人群、功能状态、屏幕验证范围、低功耗策略和发布边界。
- 统一 README 与工程文档的公开表达风格，补齐 README 中系统测试、文件系统恢复、画布素材等 HTTP 路由分类，并修正 Gitee 简介同步脚本的描述/主页字段解析方式。
- 新增 `tools/check_public_docs.py` 只读检查脚本，用于发布前核对版本号、HTTP 路由、Web 嵌入资源、分区表、release SHA256、Markdown 链接和公开文档敏感词。
- 新增 `SECURITY.md` 与 `SUPPORT.md`，明确安全报告、密钥脱敏、支持渠道、提问信息和支持边界。
- 新增 `RELEASE.md`，明确维护者发布新固件时的构建、整包合并、ZIP/SHA256、`firmware-download` 分支同步和回滚原则。
- 新增 `NOTICE.md`，集中说明 ESP-IDF、LodePNG、JPEG 解码路径、OFL 字体、Web 字体渲染和文档素材等第三方来源与维护要求。
- 增强 `tools/check_public_docs.py`，从固定文档清单改为自动扫描 Git 已追踪的公开文本文件，覆盖 `.gitee` 模板、Web 页面、硬件文档、工具脚本等更多公开内容。

## 2026-06-24

### 公开发布整理

- 明确构建缓存、SDK 配置、生成镜像和工具缓存等可再生成目录不随公开仓库发布；release、字库下载/生成脚本和必要参考素材保留在公开仓库中，字体 TTF 缓存由构建脚本按需下载到本地 ignored 目录。
- 完成公开发布一致性检查；后续发布新固件时需在 ESP-IDF v5.5.x 环境重新执行完整 `idf.py build`。
- 修正 `release/README.md` 中 `fontfs.bin` 合并地址：当前分区表 `0x920000` 是 `coredump`，`fontfs` 应烧录到 `0x940000`。
- 紧凑内置字库重新由 OFL 开源字体 `LXGW975YuanSC-400W.ttf` 生成，避免公开固件继续携带 Windows 系统字体派生产物标记。
- 同步文档中的 HTTP 路由数量、字库来源、分区表、release 固件状态和上线检查结果。

## 2026-06-04

### 新增

- 新增 Codex/中转站额度看板：配置页支持填写额度 API URL、API Key、显示单位和自动刷新间隔，墨水屏可显示余额、已用额度、今日用量、请求数与令牌数。
- 新增 `/codex_quota_config` 与 `/codex_quota_show` HTTP 接口，并将额度页纳入按键显示模式、配置备份/恢复和深睡模式记忆。

### 修复与发布

- 将 ESP-IDF main task 栈从默认值提高到 8192 字节，避免启动欢迎页切换到时钟页时出现 `task main` 栈溢出重启。
- 更新 release 整包固件与校验信息，确保 Gitee 免编译烧录包与当前源码一致。

## 2026-05-09

### 修复

- 修复时钟模式在 `clock.enabled=false` 但当前屏幕已处于时钟时不继续刷新的问题；SNTP 同步成功后会触发一次重绘，后续分钟刷新也会正常执行。
- 调整天气与时钟联动：当前为时钟模式时，天气后台刷新只通知时钟摘要更新，不再抢占成全屏天气页。
- 优化 SPIFFS 恢复流程：格式化成功后网页能拿到明确 JSON 结果并提示即将重启，设备随后自动重启离开 recovery 页面；文件系统不可用时 `/images` 返回空列表和 `spiffs_ok:false`，避免图库直接 500。
- 修复 4.2" 小屏待办显示拥挤问题：按实际高度限制条数，优先显示紧急、重要和未完成事项，页脚显示已显示/总数。
- 修复日历模式今日红圈遮挡农历、节日和节气文字的问题，改为空心圈高亮。
- 优化 SETUP 欢迎页 WiFi 信息显示，4.2" 双栏下使用 `STA/IP/AP/PW/WEB/MD` 短标签，避免热点、密码和访问地址被截断。

### 字库与显示

- 文本渲染层将全角英文、全角数字和全角常见符号映射为已有 ASCII 字形，改善网页输入法产生全角字符时的显示。
- 调整 ASCII 字体生成来源并增加校验，确保 `W/w/A-Z/a-z/0-9` 不是空白或严重裁切。
- 新增 `tools/font_extra_chars.txt` 作为小规模补字清单；后续遇到缺字时可追加字符并重新生成 `main/font_data.h`。
- 非 ASCII 字符缺字时显示占位框，不再完全空白，便于实机发现需要补充的字符。

### 工程整理

- 新增 `main/ui_theme.c` / `main/ui_theme.h`，统一墨水屏页面框架、标题栏、卡片、页脚和常用 UI 绘制。
- 将本地字体缓存目录 `tools/fonts/` 视为生成辅助资源，不随仓库提交；固件内置字库仍由 `main/font_data.h` 提供。

## 2026-05-02

### 新增

- 画布留言板增加图片预处理参数：阈值、对比度、红色敏感度、抖动强度、反色、适应/裁剪/拉伸。
- 画布图片参数支持浏览器会话内实时预览，便于提前查看黑白红墨水屏效果。
- 画布属性面板改为分组折叠，并增强图层列表、选中框、控制点、对齐辅助线和发送前确认预览。
- 新增 Gitee 二维码名片模板，默认指向 `https://gitee.com/gxp666111/miaomiao`。
- 新增 `CONTRIBUTING.md`、Gitee Issue 模板、PR 模板和发布说明模板。
- README 和工程文档补充微雪移植面板支持状态，区分主测、已验证和待实机验证。

### 修复

- 修复无 PSRAM 或内存碎片场景下部分图片转换失败的问题；在有 PSRAM 的 ESP32-S3 上优先使用 PSRAM 承载大块图片/画布数据。
- 修复画布渲染图片素材时长时间占用 CPU 导致任务看门狗报警的问题。
- 修复二维码模板在小屏上模块过小、矩阵不可靠导致手机扫码失败的问题。
- 修复 README 中 API 章节锚点在 Gitee 上无法正确跳转的问题。
- 天气请求失败日志对 URL 中的 `key=` 参数做脱敏，降低 API Key 泄露风险。
- 固定新下载源码的默认 ESP-IDF target 为 `esp32s3`，并在误设为 `esp32` 时给出明确 CMake 错误，避免 `GPIO_NUM_46` / USB Serial JTAG 头文件误报。
- 7.5 寸三色屏选项对齐 GDEW075Z08 / UC8179，并接入参考 FastFreshBWOnColor 的黑白快刷路径。

### 仓库整理

- 从公开版本库移除 Waveshare 参考包、Android App 示例源码和重复原始素材目录。
- 公开仓库保留核心固件、Web 前端、工程文档、硬件提要和 `docs/images` 展示素材。
- 将 ESP-IDF 组件清单中的最低版本约束同步为 `>=5.5.1`，与 README 和当前工程要求一致。

## 2026-04-28

### 稳定性整理

- 完成显示 epoch 仲裁，降低旧显示任务覆盖新操作的概率。
- 日历、画布、留言等异步显示接口改为等待渲染结束或返回 `canceled=true`。
- 补齐数据/媒体类 GET 接口的 Basic Auth 覆盖。
- SPIFFS 挂载失败时不再自动格式化，保留 AP/Web 恢复入口。
- 面板配置保存后提示重启，避免热切换期间 framebuffer 和缓存尺寸不一致。
- 配置备份/恢复不包含 Basic Auth 密码。
- EPD 初始化失败时进入降级诊断路径，尽量保留 Web 恢复能力。

## 2026-04-26

- 时钟任务避免每分钟全屏刷新，改为按配置和时间边界控制刷新节奏。
